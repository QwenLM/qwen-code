# 프로젝트 개요

## Qwen Code란?

**Qwen Code**는 AI 기반 명령줄 인터페이스(CLI) 도구로, 개발자가 터미널에서 직접 AI 모델과 상호작용하며 소프트웨어 개발 작업을 수행할 수 있도록 설계되었습니다.

이 프로젝트는 Google의 Gemini CLI를 기반으로 하여 Qwen3-Coder 모델에 최적화되었으며, 개발자의 생산성을 극대화하기 위한 다양한 기능을 제공합니다.

## 프로젝트 목적

Qwen Code는 다음과 같은 목적을 가지고 개발되었습니다:

1. **개발 워크플로우 자동화**: 반복적인 코딩 작업을 AI로 자동화
2. **코드 이해 향상**: 대규모 코드베이스를 빠르게 이해하고 분석
3. **AI 기반 코드 작성**: 자연어로 요청하여 코드 생성 및 리팩토링
4. **터미널 중심 워크플로우**: IDE를 벗어나 터미널에서 모든 작업 수행
5. **무료 AI 접근**: Qwen OAuth를 통해 하루 2,000회 무료 요청 제공

## 주요 기능

### 1. AI와 대화형 코드 작업

```bash
$ qwen
> 이 프로젝트의 인증 로직을 설명해줘
> UserService.ts의 login 함수를 리팩토링해줘
> 모든 컴포넌트에서 사용되지 않는 import 제거해줘
```

### 2. 파일 시스템 작업

- **파일 읽기/쓰기/편집**: AI가 자동으로 파일 조작
- **디렉토리 탐색**: 프로젝트 구조 자동 분석
- **패턴 매칭**: Glob 패턴으로 파일 검색

### 3. Shell 명령 실행

- AI가 필요한 명령어를 자동으로 실행
- 명령 실행 결과를 컨텍스트로 활용
- 안전한 샌드박스 환경 지원 (Docker/Podman)

### 4. 코드베이스 분석

- 대규모 프로젝트의 구조 자동 분석
- 의존성 그래프 추적
- 코드 품질 검토 및 개선 제안

### 5. 웹 접근 및 검색

- 최신 기술 문서 검색
- API 문서 자동 참조
- 온라인 리소스 활용

### 6. 작업 관리

- TODO 리스트 자동 생성 및 추적
- 복잡한 작업을 단계별로 분할
- 진행 상황 실시간 업데이트

## 핵심 차별화 요소

### 1. 무료 티어
- Qwen OAuth를 통해 **하루 2,000회 무료 요청**
- 별도의 API 키 없이 즉시 사용 가능

### 2. Qwen-Coder 최적화
- Qwen3-Coder 모델에 특화된 파서
- 향상된 코드 생성 품질
- 빠른 응답 속도

### 3. 비전 모델 지원
- 이미지 자동 처리
- 스크린샷 분석
- 다이어그램 이해

### 4. 서브에이전트 시스템
- 특화된 작업별 AI 에이전트
- 파일 검색, 코드 리뷰 등 전문 에이전트
- 확장 가능한 에이전트 아키텍처

### 5. 계획 모드 (Plan Mode)
- 복잡한 작업을 먼저 계획
- 단계별 실행 계획 수립
- 사용자 승인 후 실행

### 6. OpenAI API 호환성
- OpenAI SDK와 완벽 호환
- 기존 OpenAI 워크플로우 재사용 가능

## 사용 사례

### 코드 리팩토링
```bash
$ qwen "이 프로젝트의 모든 클래스 컴포넌트를 함수 컴포넌트로 변환해줘"
```

### 버그 수정
```bash
$ qwen "UserController에서 발생하는 null pointer 에러 찾아서 수정해줘"
```

### 테스트 작성
```bash
$ qwen "AuthService의 모든 함수에 대한 단위 테스트 작성해줘"
```

### 문서 생성
```bash
$ qwen "API 엔드포인트에 대한 README.md 작성해줘"
```

### 코드베이스 이해
```bash
$ qwen "이 프로젝트의 인증 플로우를 다이어그램으로 설명해줘"
```

## 지원 환경

- **운영체제**: Linux, macOS, Windows (WSL)
- **Node.js**: ≥20.0.0
- **IDE 통합**: VS Code, Zed
- **컨테이너**: Docker, Podman

## 프로젝트 역사

- **기반**: Google Gemini CLI
- **최적화**: Qwen3-Coder 모델
- **현재 버전**: 0.2.0
- **라이선스**: Apache 2.0
- **개발**: QwenLM 팀
